Payment method

Orders are logged to a private Google Sheet. Payment secrets live in Google Apps Script properties, not in this public repo.

How The Payment Flow Works

1. Sheet record
The Apps Script writes the order to Google Sheets first, so there is a trace even if payment is manual.

2. Stripe Checkout
For card or wallet payments, Apps Script creates the Stripe Checkout Session and redirects the buyer there.

3. Manual options
PayPal and PayID / bank transfer can be recorded in the same sheet and confirmed manually before delivery.

4. Delivery
Downloads can be delivered after payment confirmation by email, Drive link, or a later automated file link.
