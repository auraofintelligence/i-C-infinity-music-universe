# Infinity Engine Studio Builders

The builder pages in this folder create agent-ready Markdown for image and video production.

The songs already exist on the main song pages. Start with `001 Human Listen` when a person wants to reverse-engineer the song by listening, tapping signals, shaping dials, and exporting SRT/Markdown cues. Use `007 Ingestion` to clean that pass into lyrics, audio, stems, emotion, tone, pitch, beats, metadata, and target-query notes. Later builders turn that profile into visual assets, video clips, distribution-fit plans, reviews, and handoffs.
