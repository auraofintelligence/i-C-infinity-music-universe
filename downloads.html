Packaging Principle

A paid download should be a direct value exchange, not charity language and not a vague tip jar. The buyer gets useful files. Luke gets cash to keep building.

Free streaming can stay as discovery. Paid downloads should feel like owning a clean, well-labelled slice of the project.

Packaging Checklist

1. Choose the bundle names and prices.

2. Decide file formats: MP3, WAV, FLAC, MP4, lyrics PDF, cover art, and notes.

3. Build one folder per bundle with simple names and no local machine paths.

4. Write buyer-facing copy only after the actual contents are clear.
